Grayscale v1.1.1
By Sriranga Veeraraghavan <ranga@calalum.org>

Graysale is a menubar application to toggle the display between grayscale
and color mode, enable or disable night shift (and set its strength), set 
the main display's brightness, and enable or disable dark mode.

To install:

Copy Grayscale.app to /Applications

Supported MacOSX versions:

v. 1.1.0+ - 10.14+
v. 1.0.1  - 10.9+
v. 1.0.0  - 10.6+
